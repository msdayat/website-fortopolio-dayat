About Me

Halo! Nama saya Muhamad Syamsul Hidayat. Saya seorang pengembang web yang bersemangat dengan minat dalam membuat situs web yang dinamis dan ramah pengguna. Selama bertahun-tahun, saya telah mengasah keterampilan saya dalam HTML, CSS, JavaScript, dan berbagai kerangka pengembangan web untuk membangun aplikasi web yang fungsional dan estetis..

Perjalanan saya memasuki dunia web development bermula dari rasa penasaran untuk memahami cara kerja website. Rasa ingin tahu ini dengan cepat berubah menjadi passion, menuntun saya untuk menekuni karir di bidang ini. Saya mulai dengan mempelajari dasar-dasar HTML dan CSS, dan secara bertahap memperluas pengetahuan saya dengan memasukkan JavaScript dan kerangka kerja populer seperti Bootstrap.
